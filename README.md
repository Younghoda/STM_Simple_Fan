# STM_Simple_Fan

-정지, 1단, 2단, 3단의 총 4개의 버튼을 이용해서 선풍기의 속도를 조절할 수 있도록 한다.(-PWM을 조절하여 선풍기의 속도를 조절)

-누른 버튼에 따라 각각의 LED가 켜지고 LCD 화면에 현재 선풍기의 속도를 표시한다.
